# Reset Password

Terdapat dua cara untuk melakukan reset password pada sistem Berumrah:

1. **Melalui Halaman Login**  
   Kunjungi halaman:  
   **https://hisar.berumrah.id/auth/login**

2. **Melalui Reset Password Manual (OTP WhatsApp)**  
   Anda dapat melakukan reset password dengan mengirimkan kode OTP ke WhatsApp Anda.

## Langkah-langkah Reset Password

1. **Buka Halaman Login**  
   Pastikan Anda berada di halaman **Login**, lalu klik tombol **"Lupa Password?"**.

2. **Masukkan Nomor Telepon**  
   Pada tampilan yang muncul, masukkan nomor telepon yang terdaftar pada akun Anda, kemudian tekan tombol **"Kirim Kode Reset"**.

3. **Terima Kode Reset via WhatsApp**  
   Sistem akan mengirimkan kode OTP berisi **4 digit** melalui WhatsApp.  
   Gunakan kode tersebut untuk melanjutkan proses reset password.

4. **Masukkan Kode OTP & Buat Password Baru**  
   - Masukkan **4 digit kode** dari WhatsApp.  
   - Isi **password baru** dan **konfirmasi password**.  
   - Tekan tombol **"Reset Password"** untuk menyelesaikan proses.

---

Proses reset password selesai setelah Anda menerima notifikasi bahwa password telah berhasil diperbarui.

# Pemesanan Program

Panduan ini menjelaskan cara memilih program perjalanan, melakukan pemesanan, hingga menerima konfirmasi dari sistem dan melanjutkan ke proses pembayaran.

## 1. Masuk ke Halaman Program
Setelah berhasil login, buka menu **“PROGRAM”** pada sidebar utama.

Pada halaman ini Anda akan melihat berbagai paket perjalanan yang tersedia, seperti:
- Umrah Reguler  
- Umrah Plus  
- Haji Khusus  

Setiap paket menampilkan ringkasan informasi seperti:
- Nama program  
- Harga  
- Tanggal keberangkatan  

## 2. Pilih Program yang Diinginkan
Pilih program yang sesuai kebutuhan Anda, lalu klik tombol **“Pesan Sekarang”** pada bagian bawah kartu program.  
Tombol ini akan mengarahkan Anda ke halaman detail program.

## 3. Halaman Detail Program
Pada halaman detail program, Anda dapat melihat informasi lengkap seperti:
- Harga dan fasilitas yang disediakan  
- Jadwal keberangkatan serta durasi perjalanan  
- Informasi hotel, maskapai, dan layanan tambahan  

Pastikan membaca seluruh detail dengan teliti.  
Jika sudah yakin, klik tombol **“Lanjutkan Pemesanan”** untuk melanjutkan.

## 4. Isi Data Jamaah
Anda akan diarahkan ke halaman **“Data Jamaah”**. Isikan seluruh informasi diri dengan lengkap, seperti:
- Nama lengkap  
- Nomor telepon  
- Data lain yang diperlukan  

Jika sudah selesai, klik tombol **“Lanjutkan”**.

## 5. Pemesanan Berhasil Diterima
Setelah proses konfirmasi selesai, sistem akan menampilkan notifikasi bahwa pemesanan berhasil diterima.  
Hal ini menandakan bahwa pesanan Anda telah masuk dan sedang diproses oleh tim admin atau reservasi.

## 6. Terima Konfirmasi Melalui WhatsApp
Beberapa saat setelah itu, Anda akan menerima pesan dari WhatsApp resmi **Hisar Global** berisi:
- Informasi bahwa pemesanan telah diterima  
- Rincian program yang Anda pesan  
- Akun login (username dan password)  

Gunakan data login tersebut untuk masuk ke sistem.

## 7. Masuk ke Akun Jamaah
Masukkan **username** dan **password** yang dikirim melalui WhatsApp, lalu klik **“Login”**.

Setelah berhasil login, Anda akan langsung diarahkan ke halaman yang menampilkan **Detail Transaksi**.

## 8. Lanjut ke Menu Pembayaran
Pada halaman detail transaksi, ditampilkan:
- Total biaya program  
- Rincian harga  
- Status pembayaran  

Dari halaman ini Anda dapat melanjutkan ke menu pembayaran untuk menyelesaikan transaksi sesuai instruksi yang tersedia.

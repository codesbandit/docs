# Tambah Kategori Paket

Panduan ini menjelaskan cara menambahkan kategori paket baru di sistem Berumrah ERP.

## Langkah-langkah

1. **Masuk ke Modul “Paket”**  
   Pada menu utama, pilih modul **Paket**.

2. **Pilih Submodul “Kategori Paket”**  
   Setelah modul Paket terbuka, klik submodul **Kategori Paket** untuk melihat daftar kategori yang ada.

3. **Klik Tombol “Tambah Kategori Paket”**  
   Setelah halaman daftar kategori paket muncul, klik tombol **Tambah Kategori Paket** untuk membuat kategori baru.

---

Setelah tombol diklik, Anda akan diarahkan ke formulir untuk mengisi detail kategori paket baru.

# Login ke Dashboard

Ikuti langkah-langkah berikut untuk masuk ke halaman dashboard akun Anda.

## 1. Kunjungi Halaman Login
Buka halaman login melalui link berikut:  
**https://hisar.berumrah.id/auth/login**

## 2. Masukkan Email dan Password
Isikan **email** dan **password** yang terdaftar pada akun Anda.  
Pastikan keduanya dimasukkan dengan benar.

## 3. Klik Tombol LOGIN
Setelah email dan password sudah terisi, klik tombol **“LOGIN”** untuk masuk ke halaman dashboard.

Anda akan diarahkan ke dashboard utama jika data login benar dan akun Anda aktif.

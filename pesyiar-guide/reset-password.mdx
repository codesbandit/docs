# Reset Password

Terdapat dua cara untuk melakukan reset password pada sistem Berumrah:

1. **Melalui Halaman Login**  
   Kunjungi halaman berikut:  
   **https://hisar.berumrah.id/auth/login**

2. **Melalui Reset Password Manual (OTP WhatsApp)**  
   Anda dapat melakukan reset password dengan mengirimkan kode OTP melalui WhatsApp.

## Langkah-langkah Reset Password Manual

1. **Buka Halaman Login**  
   Pastikan Anda berada di halaman **Login**, lalu klik tombol **“Lupa Password?”**.

2. **Masukkan Nomor Telepon**  
   Pada tampilan yang muncul, masukkan nomor telepon akun Anda, kemudian tekan tombol **“Kirim Kode Reset”**.

3. **Terima Kode Reset via WhatsApp**  
   Anda akan menerima notifikasi berisi **4 digit kode reset** melalui WhatsApp.  
   Gunakan kode tersebut untuk melanjutkan proses reset password.

4. **Masukkan Kode OTP & Password Baru**  
   - Masukkan **4 digit kode OTP**.  
   - Isi **password baru** dan **konfirmasi password**.  
   - Tekan tombol **“Reset Password”** untuk menyelesaikan proses.

---

Setelah berhasil, password Anda akan diperbarui dan Anda dapat login menggunakan password yang baru.

## Hapus Paket

Ikuti langkah berikut untuk menghapus atau menonaktifkan paket:

1. **Masuk ke Modul Paket**  
   Pergi ke modul **“Paket”**, kemudian pilih submodul **“Paket”**.

2. **Pilih Paket yang Akan Dihapus**  
   Setelah halaman daftar paket muncul, cari paket yang ingin dihapus.

3. **Hapus Paket**  
   Klik tombol **“Hapus”** pada baris data paket tersebut.  
   Sistem akan meminta konfirmasi sebelum paket dihapus.

---

## Menonaktifkan Paket (Alternatif Selain Hapus)

Jika tidak ingin menghapus paket secara permanen, Anda dapat menonaktifkannya:

1. Buka detail paket yang ingin dinonaktifkan.  
2. Klik tombol di sebelah teks **“Status”** untuk mengubah status menjadi *Tidak Aktif*.  
3. Setelah itu, klik tombol **“Simpan”** pada tab **Penghargaan** untuk menyimpan perubahan.

## Tambah Paket

Ikuti langkah-langkah berikut untuk menambahkan paket baru:

1. **Masuk ke Modul Paket**  
   Pergi ke modul **“Paket”**, kemudian pilih submodul **“Paket”**.

2. **Buka Form Tambah Paket**  
   Setelah halaman daftar paket muncul, klik tombol **“Tambah Paket”**.  
   Sistem akan menampilkan form untuk membuat paket baru.

3. **Isi Form Paket Baru**  
   Lengkapi kolom berikut dengan benar:
   - **Judul Paket** — masukkan nama atau judul paket yang ingin dibuat.  
   - **Slug (Opsional)** — alamat singkat untuk URL (otomatis atau bisa disesuaikan).  
   - **Kategori Paket** — pilih kategori paket yang sesuai.  
   - **Deskripsi** — tuliskan penjelasan lengkap mengenai detail paket.  
   - **Tipe Kamar** — tentukan jenis kamar (Double, Triple, Quad).  
   - **Layanan Tambahan** — tambahkan layanan tambahan jika tersedia.

4. **Simpan Paket Baru**  
   Jika semua data telah terisi, klik tombol **“Simpan”** untuk menambahkan paket baru.

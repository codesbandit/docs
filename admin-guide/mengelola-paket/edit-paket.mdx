## Edit Paket

Untuk mengedit paket yang sudah ada, ikuti langkah-langkah berikut:

1. **Masuk ke Modul Paket**  
   Pergi ke modul **“Paket”**, kemudian pilih submodul **“Paket”**.

2. **Pilih Paket yang Akan Diedit**  
   Setelah halaman daftar paket muncul, cari paket yang ingin diperbarui.  
   Klik tombol **“Edit”** pada baris paket tersebut.

3. **Perbarui Data Paket**  
   Sistem akan menampilkan form edit paket.  
   Perbarui kolom sesuai kebutuhan:
   - **Judul Paket** — ubah nama atau judul paket jika diperlukan.  
   - **Slug (Opsional)** — sesuaikan alamat singkat untuk URL (otomatis atau manual).  
   - **Kategori Paket** — pilih atau ubah kategori yang sesuai.  
   - **Deskripsi** — revisi atau lengkapi penjelasan mengenai detail paket.

4. **Simpan Perubahan**  
   Setelah semua pembaruan dilakukan, klik tombol **“Simpan Perubahan”** untuk menyimpan update paket.

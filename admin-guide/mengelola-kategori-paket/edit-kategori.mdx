## Edit Kategori Paket

Untuk mengedit kategori paket yang sudah ada, ikuti langkah berikut:

1. **Masuk ke Modul Paket**  
   Pergi ke modul **“Paket”**, kemudian pilih submodul **“Kategori Paket”**.

2. **Pilih Kategori yang Akan Diedit**  
   Setelah halaman daftar kategori paket muncul, cari kategori yang ingin diperbarui.  
   Klik tombol **“Edit”** pada baris kategori tersebut.

3. **Perbarui Data Kategori**  
   Sistem akan menampilkan form edit kategori.  
   Perbarui kolom yang diperlukan:
   - **Nama Kategori** — ubah nama kategori paket jika diperlukan.  
   - **Slug Kategori** — sesuaikan alamat singkat untuk URL (otomatis atau manual).  
   - **Deskripsi** — lengkapi atau revisi penjelasan kategori.  
   - **Gambar** — unggah gambar baru jika ingin mengganti gambar sebelumnya.  
   - **Status** — pilih **Aktif** jika kategori digunakan, atau **Tidak Aktif** jika tidak ingin ditampilkan.

4. **Simpan Perubahan**  
   Setelah selesai memperbarui data, klik tombol **“Simpan Perubahan”** untuk menyimpan update kategori.

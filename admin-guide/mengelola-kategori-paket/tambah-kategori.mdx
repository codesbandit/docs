## Tambah Kategori Paket

Ikuti langkah berikut untuk menambahkan kategori paket baru:

1. **Masuk ke Modul Paket**  
   Pergi ke modul **“Paket”**, kemudian pilih submodul **“Kategori Paket”**.

2. **Buka Form Tambah Kategori**  
   Setelah halaman daftar kategori paket muncul, klik tombol **“Tambah Kategori Paket”**.  
   Sistem akan menampilkan form untuk menambahkan kategori baru.

3. **Isi Form Kategori Paket**  
   Lengkapi data berikut dengan benar:
   - **Nama Kategori** — nama kategori paket yang ingin dibuat.  
   - **Slug Kategori** — alamat singkat untuk URL (otomatis atau bisa disesuaikan).  
   - **Deskripsi** — penjelasan singkat mengenai kategori paket.  
   - **Gambar** — unggah gambar pendukung untuk kategori paket.  
   - **Status** — pilih **Aktif** jika kategori digunakan, atau **Tidak Aktif** jika belum ingin ditampilkan.

4. **Simpan Data**  
   Jika semua data telah terisi dengan benar, klik tombol **“Simpan”** untuk menambahkan kategori paket baru.

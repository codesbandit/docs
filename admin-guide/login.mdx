## Masuk ke Halaman Dashboard

Untuk dapat masuk ke halaman dashboard, ikuti langkah-langkah berikut:

1. **Kunjungi halaman login**  
   Buka link: `https://dash.berumrah.id/signin`

2. **Masukkan kredensial akun**  
   Input **email atau nomor WhatsApp** serta **password** Anda.  
   Pastikan data yang dimasukkan sudah benar.

3. **Klik tombol “SIGN IN”**  
   Jika email/nomor WhatsApp dan password sesuai, Anda akan diarahkan ke halaman dashboard.

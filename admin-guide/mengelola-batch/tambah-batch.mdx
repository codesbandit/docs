## Tambah Batch

Ikuti langkah-langkah berikut untuk membuat batch baru:

1. **Masuk ke Modul Batch**  
   Pergi ke modul **“Paket”**, lalu pilih submodul **“Batch”**.

2. **Buka Form Tambah Batch**  
   Setelah halaman daftar batch muncul, klik tombol **“Tambah Batch”**.  
   Sistem akan menampilkan form untuk menambahkan batch baru.

3. **Isi Form Batch Baru**  
   Lengkapi kolom berikut sesuai kebutuhan:
   - **Paket** — pilih paket induk untuk batch ini.  
   - **Nama Batch** — isi nama batch yang akan dibuat.  
   - **Cover Image Batch** — unggah gambar untuk batch.  
   - **Tampilkan sebagai Featured Batch** — centang jika ingin tampil di halaman utama.  
   - **Tanggal Berangkat** — pilih tanggal keberangkatan.  
   - **Tanggal Pulang** — pilih tanggal kepulangan.  
   - **Durasi (Hari)** — dihitung otomatis dari tanggal berangkat dan tanggal pulang.  
   - **Status** — pilih *Open*, *Full*, atau *Closed*.  
   - **Harga** — masukkan harga utama paket.  
   - **Harga Coret** — harga sebelum diskon (opsional).  
   - **Kuota** — jumlah maksimum peserta.  
   - **Sisa Kursi** — terisi otomatis berdasarkan sisa ketersediaan kursi.  
   - **Komisi Flat** — komisi tetap per transaksi.  
   - **Komisi Persen (%)** — komisi dalam bentuk persentase.  
   - **Layanan / Fasilitas (Opsional)** — masukkan fasilitas tambahan jika ada.  
   - **Batas Booking** — tanggal terakhir pemesanan batch.  
   - **Catatan** — isi informasi atau keterangan tambahan jika diperlukan.

4. **Simpan Batch**  
   Setelah semua data terisi, klik tombol **“Simpan Batch”** untuk menyimpan batch baru.

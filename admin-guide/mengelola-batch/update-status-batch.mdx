## Update Status Batch

Untuk mengubah status batch, ikuti langkah berikut:

1. **Masuk ke Modul Batch**  
   Pergi ke modul **“Paket”**, lalu pilih submodul **“Batch”**.

2. **Cari Batch yang Akan Diubah Statusnya**  
   Setelah halaman daftar kanban batch muncul, temukan batch yang ingin diperbarui.

3. **Ubah Status Batch**  
   Klik dan **geser (drag & drop)** batch ke kolom status yang diinginkan:  
   - **Open**  
   - **Full**  
   - **Closed**

   Perubahan status akan tersimpan otomatis sesuai posisi baru.

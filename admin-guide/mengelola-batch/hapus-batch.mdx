## Hapus Batch

Ikuti langkah-langkah berikut untuk menghapus batch:

1. **Masuk ke Modul Batch**  
   Pergi ke modul **“Paket”**, lalu pilih submodul **“Batch”**.

2. **Pilih Batch yang Akan Dihapus**  
   Setelah halaman daftar kanban batch muncul, cari data batch yang ingin dihapus.

3. **Hapus Batch**  
   Klik tombol **“Hapus”** pada baris data batch tersebut.  
   Sistem akan menampilkan popup konfirmasi.

4. **Konfirmasi Penghapusan**  
   Klik **“Ya, Hapus”** untuk menghapus batch secara permanen dari daftar.

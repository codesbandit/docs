## Edit Batch

Untuk memperbarui data batch, ikuti langkah-langkah berikut:

1. **Masuk ke Modul Batch**  
   Pergi ke modul **“Paket”**, lalu pilih submodul **“Batch”**.

2. **Pilih Batch yang Akan Diedit**  
   Setelah halaman daftar kanban batch muncul, cari batch yang ingin diperbarui.  
   Klik tombol **“Edit”** pada batch tersebut.

3. **Perbarui Data Batch**  
   Sistem akan menampilkan form **Edit Batch**.  
   Perbarui kolom sesuai kebutuhan:
   - **Paket** — ubah paket induk jika diperlukan.  
   - **Nama Batch** — sesuaikan nama batch.  
   - **Cover Image Batch** — unggah gambar baru jika ingin mengganti gambar lama.  
   - **Tampilkan sebagai Featured Batch** — centang jika ingin ditampilkan di halaman utama.  
   - **Tanggal Berangkat & Tanggal Pulang** — sesuaikan jadwal keberangkatan dan kepulangan.  
   - **Durasi (Hari)** — otomatis dihitung berdasarkan tanggal berangkat dan pulang.  
   - **Status** — pilih *Open*, *Full*, atau *Closed* sesuai kondisi.  
   - **Harga & Harga Coret** — sesuaikan harga utama dan harga promo.  
   - **Kuota & Sisa Kursi** — ubah kapasitas maksimal dan jumlah kursi yang tersedia.  
   - **Komisi Flat / Komisi Persen (%)** — sesuaikan komisi untuk agen.  
   - **Layanan / Fasilitas (Opsional)** — ubah atau tambahkan fasilitas tambahan.  
   - **Batas Booking** — sesuaikan batas terakhir pemesanan.  
   - **Catatan** — tambahkan atau ubah informasi tambahan yang diperlukan.

4. **Simpan Perubahan**  
   Setelah semua pembaruan selesai dilakukan, klik tombol **“Simpan Perubahan”** untuk menyimpan perubahan data batch.
